# 모바일 청첩장 새 프로젝트

## 1. 사진 넣기
`images` 폴더에 아래 이름으로 사진을 넣으세요.

- hero.jpg : 첫 화면 세로 사진
- couple.jpg : 초대글 아래 대표 사진
- gallery-01.jpg ~ gallery-06.jpg : 갤러리 사진
- closing.jpg : 마지막 화면 사진
- og-image.jpg : 카카오톡 공유 썸네일

권장 크기:
- hero.jpg / closing.jpg: 세로 1200 × 1800 이상
- gallery: 긴 변 1600px 이상
- og-image.jpg: 1200 × 630

## 2. 음악 넣기
`music/bgm.mp3` 파일을 넣으세요.

## 3. 반드시 수정할 정보
`index.html`에서 검색하여 수정:
- 부모님 성함
- 전화번호
- 계좌번호
- 지하철/주차 안내

## 4. 카카오 공유
`js/kakao.js`의 `KAKAO_JAVASCRIPT_KEY`에 카카오 개발자센터 JavaScript 키를 입력하세요.

## 5. 실행
VS Code에서 `index.html`을 우클릭 → Open with Live Server.
